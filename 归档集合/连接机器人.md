# 连接机器人

环境假设，如果我们和其他比赛选手处在同一个比赛场地，相同网段；我们可以通过机器人身上的名称来确定其IP，通过pad连接指定机器人即可查看IP。之后使用VNC连接即可实现文件共享。

# 准备工作

显示器连接上机器人并登陆后，应该首先打开机器人的API文档，在机器人系统浏览器输入127.0.0.1:9090/v1/ui查看，方便调用API控制机器人。
